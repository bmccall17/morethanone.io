# v0.0.9 - admin panel & RCV World article import

**released:** feb 22, 2026

## summary
Full admin panel with CMS for content, FAQs, templates, playtest feedback, and keepalive monitoring. RCV World tracker with Gemini-powered article import pipeline, Readability-based HTML extraction, and step-by-step diagnostics for observability.

---

## changes
- Admin panel with sidebar navigation at `/admin` — content sections, FAQs, templates, playtest feedback, keepalive, RCV World, metrics
- Admin bearer token auth system (`useAdminAuth` hook + `verifyAdminAuth` middleware)
- RCV World CRUD — create/edit/list/delete real-world RCV examples with Supabase
- Article import pipeline — fetch URL, extract text with Readability + linkedom, AI field extraction via Gemini 2.0 Flash
- Import pipeline diagnostics — `diagnostics` trace object in API responses + `[import]` server logs at every failure point
- Browser-side debug info — collapsible `<details>` block in import UI showing full diagnostics trace
- Content sections CMS (CRUD for "How this works" page content)
- FAQ management with admin CRUD + public `/faq` page
- Template management (moved from hardcoded `POLL_TEMPLATES` to DB-backed `templates` table)
- Playtest feedback pipeline with "promote to FAQ" workflow
- Keepalive cron endpoint with audit trail logging
- Public RCV World page at `/rcv-world`
- Public API routes for content, FAQs, templates, RCV World
- Vercel deployment URL update
- Admin cheatsheet documentation
- New dependencies: `@mozilla/readability`, `linkedom`
