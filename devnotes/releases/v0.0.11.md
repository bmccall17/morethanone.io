# v0.0.11 - usage metrics on admin dashboard

**released:** feb 23, 2026

## summary
Adds a "Usage & Costs" section to the admin dashboard with visibility into Gemini API token usage, function activity (events as proxy for Vercel invocations), and Supabase database size — early warning before costs spike on the Hobby plan.

---

## changes
- New `usage_log` table for tracking API usage (Gemini tokens, function calls)
- New `get_table_sizes()` Postgres RPC function for querying database sizes
- `logUsage()` fire-and-forget utility mirroring the `trackEvent()` pattern
- Gemini token usage automatically logged on every RCV World article import
- New `/api/admin/usage` admin API returning three data sections: gemini tokens, event activity, and DB table sizes
- `UsagePanel` component with StatCards and BarCharts for all three cost dimensions
- Gemini section: total calls, total tokens, estimated cost, tokens per call, calls-per-day chart
- Function activity section: 7d/30d/all-time event counts, events-by-type chart
- Supabase section: total DB size, percentage of 500MB Hobby limit, table size breakdown
