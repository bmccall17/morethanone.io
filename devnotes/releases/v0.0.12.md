# v0.0.12 - real election demos + nav polish + security hardening

**released:** mar 11, 2026

## summary
Enriches the demo experience with real-world ranked choice election data (Maine 2018 governor, Alaska 2022 special), improves scenario realism across all 5 demos, adds a unified Learn navigation bar across demo/FAQ/RCV World pages, hardens Supabase RLS policies for `rcv_world_examples`, and fixes keepalive endpoint issues.

---

## changes
- Added Maine 2018 gubernatorial election as a new demo scenario with real candidate names, ballot distributions, and related links
- Added Alaska 2022 special congressional election as a new demo scenario with real data
- Improved scenarios 1–5 with realistic ballot distributions and real-number participant counts
- New `LearnNav` component providing sticky tabbed navigation (Demo / FAQ / RCV World) with Host/Join CTAs across all learn pages
- Integrated LearnNav into demo, FAQ, RCV World, and homepage "How it works" section
- Added RLS policy for `rcv_world_examples` table — public read-only, writes restricted to service_role
- Migrated admin RCV World API routes to use service-role Supabase client for secure writes
- Created `src/lib/supabase/admin.ts` — server-side admin client using `SUPABASE_SERVICE_ROLE_KEY`
- Fixed keepalive endpoints (both admin and public) — resolved connection/response issues
- Updated README.md with current project description
