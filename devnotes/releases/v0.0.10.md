# v0.0.10 - admin test rounds & content polish

**released:** feb 22, 2026

## summary
Admin can now host test rounds from the dashboard that behave identically to real rounds but are invisible to analytics, public listings, and metrics. Also includes /rcv-world and /faq navigation improvements, related-items content linking, and Gemini model updates.

---

## changes
- Admin test rounds — `is_test` flag on rounds table, skips all 6 analytics events across the round lifecycle
- Test rounds excluded from `/api/rounds/public` listing and admin metrics dashboard
- Admin dashboard quick-action buttons: "Host test round" and "Join round"
- Host create page `?test=1` param: auto-sets private, passes `is_test`, shows "Test round" badge
- Replay propagation — replaying a test round creates another test round
- `/rcv-world` and `/faq` back navigation improvements
- Related items content linking system (`RelatedItemsPicker`, `RelatedLinks` components, `related-items` utility)
- RCV World content types migration (`content_types` column)
- Resource type improvements for admin content
- Gemini model update to 2.5-flash with increased timeout
